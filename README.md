### 设计思想

基于 vue3 和 typescript，仅仅依赖 `lodash^4.17.21`。

特性：

- 流程图编辑器 xprocess-editor
- 图形注册机制
- 画布插件机制
- UI 组件
- 系统独立对接
- 导入导出
- 独立复制成图片分享
